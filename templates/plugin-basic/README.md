# {{PLUGIN_NAME}}

A basic plugin for youtube-music-cli.

## Installation

```bash
youtube-music-cli plugins install <path-to-plugin>
```

## Features

- Logs when a track starts playing
- Logs when playback is paused
- Logs when the track changes

## Configuration

This plugin has no configuration options.

## Development

1. Clone this repository
2. Modify the `index.ts` file
3. Test with `youtube-music-cli plugins install .`
