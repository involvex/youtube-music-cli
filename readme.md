# youtube-music-cli

- A Commandline music player for youtube-music

## Install

```bash
$ npm install --global @involvex/youtube-music-cli
```

## CLI

```
$ youtube-music-cli --help

  Usage
    $ youtube-music-cli

  Options
    --name  Your name (to be changed)

  Examples
    $ youtube-music-cli --name=Jane
    Hello, Jane
```
